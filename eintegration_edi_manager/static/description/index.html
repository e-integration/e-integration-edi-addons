Send invoices and quotations via EDI

Create EDI documents from a business object by clicking the "Send by EDI" button. All EDI documents associated with the particular business object are listed under the "EDI Documents" tab page.

Create several EDI documents at one go using the "Send by EDI" multi action.

Customize EDI documents per business object

Select the relationships which will be stored in the created EDI document. Every business object has its own template.

Manage EDI partners

Create and manage multiple EDI document partner records with different GLNs per contact. Those records serve as senders or recipients of EDI documents.
